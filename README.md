# Music App

Backend REST API for a music and podcast streaming application built with Spring Boot, PostgreSQL, Redis and Apache Kafka. The system includes a Python ML service for audio analysis, behavioral recommendations and mood detection.

## Projects

- `music-app-api` — Spring Boot REST API (Java 21)
- `music-ml-service` — ML service (Python 3.12, FastAPI, PyTorch)
- `music-app-mobile` — Android mobile app (Kotlin)
- `music-app-desktop` — Desktop app (C#, .NET MAUI, Windows)

## Tech Stack

- Java 21, Spring Boot 3.4.4
- Python 3.12, FastAPI, PyTorch, scikit-learn
- PostgreSQL, Apache Kafka, Redis, Apache Cassandra
- Kubernetes (AKS), Docker, Helm, ArgoCD
- Azure Blob Storage, Azure CDN
